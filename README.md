This project predicts medical insurance charges using supervised machine learning algorithms.
Medical Cost Personal Dataset

Features:

- Age
- Gender
- BMI
- Children
- Smoker
- Region

Target:

- Insurance Charges
-  Models Used

- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor

5. Feature Scaling
6. Model Training
7. Model Evaluation
8. Prediction

 Evaluation Metrics
  MAE
- MSE
- RMSE
- R² Score

Results

Random Forest Regressor achieved the best performance among all implemented models.
